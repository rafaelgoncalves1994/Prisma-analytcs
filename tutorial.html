TUTORIAL DE CHAVE API

Como criar sua API Key no Google Gemini

Siga os passos abaixo para gerar sua chave de API utilizando o Google AI Studio:

  1. Acesse o Google AI Studio pelo link: https://aistudio.google.com/
  2. Faça login com sua conta Google.
  3. Se for seu primeiro acesso, aceite os termos de privacidade exibidos pela plataforma.
  4. No painel inicial, clique em Get API Key (ou Obter API Key).
  5. Você verá uma lista com chaves já criadas (se houver). Para criar uma nova, clique em Criar Chave de API.
  6. O sistema solicitará a criação de um projeto. Clique em Criar Projeto.
  7. Escolha um nome para o projeto. Exemplos:
    • ProjetoCiencia
    • ProjetoCienciaDeDados
    (Não utilize acentos ou caracteres especiais.)
  8. Após confirmar, o projeto será criado e sua API Key será gerada automaticamente.
  9. Clique em Copiar para copiar a chave.
  10. Cole a API Key no sistema, aplicação ou ambiente em que você planeja utilizá-la.

Sua API Key é privada nunca compartilhe com terceiros ou publique em repositórios públicos.
